# To Do List (Console version)

## 조건
* ToDo 상태로 할 일을 추가한다.
* 상태 값에 상과없이 삭제한다.
* 조회할 때, 상태 값을 기준으로 출력한다.
* class는 한 가지 역할을 가져야한다.
* method는 한 가지 책임만 가져야한다.

## 학습 포인트
* Clean Code
* Collection - ArryaList
* Generic
* Java Beans Pattern
* Java MVC Pattern
